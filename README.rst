Splitter for speedruns

Installation
------------

Install via pip::

    pip install fizzure

Development
-----------

Run test suite to ensure everything works::

    make test

Release
-------

To publish your plugin to pypi, sdist and wheels are registered, created and uploaded with::

    make release
